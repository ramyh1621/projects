# Medical appointment management 
# About the project :
The goal of this tab is to develop a C program to manage patien appointment in a medical clinic each patien characterized by :

 * A file numbere (ID).
 * A name.
 * A phone number.
 * A date of birth.
 * An appointment date.
 * An appointment time (hour and minute).


# Main Program (C) :
Implement a menu-driven program to test all functionalities:

1) Input patients.
2) Display patients.
3) Sort patients.
4) Add a patient.
5) Delete a patient.
6) Search for a patient.
7) Count appointments by date.
8) Archive (save) patients in a structured text file (you can find a text file in the branch to see how does this work). 
0) Exit
